# ProyectoFinalCailotto

Simulador de Ecommerce en JavaScript.

## Funcionalidad
- Carga productos desde un archivo JSON.
- Renderiza tarjetas dinámicas en HTML.
- Permite agregar productos al carrito.
- Calcula el total y simula el checkout con SweetAlert2.

## Tecnologías
- HTML
- CSS
- JavaScript
- SweetAlert2

## Ejecución
Abrir `index.html` en el navegador.
